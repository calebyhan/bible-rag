## 프로젝트 개요

### Bible RAG란?

Bible RAG는 인공지능 기반의 **의미 검색(Semantic Search)**을 활용한 성경 공부 플랫폼입니다. 단순한 키워드 검색을 넘어서, 여러분이 찾고자 하는 **의미**를 이해하고 관련된 성경 구절을 찾아줍니다.

### 왜 특별한가요?

기존 성경 앱들은 정확한 단어를 입력해야만 검색이 가능했습니다. 예를 들어:
- ❌ "사랑"으로 검색 → "사랑"이라는 단어가 들어간 구절만 찾음
- ✅ Bible RAG로 검색 → 사랑, 애정, 자비, 긍휼 등 관련 개념의 구절을 모두 찾음

**핵심 차별점:**
1. 🧠 **의미 이해**: AI가 문맥과 의미를 파악
2. 🇰🇷 **한국어 최적화**: 한국어를 위한 특별한 기능들
3. 📖 **원어 통합**: 히브리어, 헬라어 원문 확인 가능
4. 🔗 **자동 교차참조**: 관련 구절 자동 연결
5. 💰 **무료**: 완전 무료, 오픈소스

## 주요 기능

### 1. 의미 검색 (Semantic Search)

**자연스러운 질문으로 검색하세요:**

```
"예수님이 용서에 대해 하신 말씀"
→ 마태복음 6:14-15, 누가복음 6:37, 마태복음 18:21-22
```

```
"어려울 때 위로받는 말씀"
→ 시편 23편, 마태복음 11:28-30, 고린도후서 1:3-4
```

**키워드가 정확히 일치하지 않아도 됩니다!**

### 2. 다국어 번역 지원

**한국어 번역본:**
- 개역개정
- 개역한글
- 새번역
- 공동번역

**영어 번역본:**
- NIV (New International Version)
- ESV (English Standard Version)
- NASB (New American Standard Bible)

**원어:**
- 히브리어 (구약)
- 헬라어 (신약)
- 아람어 (일부)

### 3. 병렬 번역 보기

여러 번역본을 한눈에 비교:

| 요한복음 3:16 | |
|---|---|
| **개역개정** | 하나님이 세상을 이처럼 사랑하사<br/>독생자를 주셨으니 |
| **새번역** | 하나님께서 세상을 이처럼 사랑하셔서<br/>외아들을 내주셨습니다 |
| **영어-NIV** | For God so loved the world that he<br/>gave his one and only Son |

### 4. 원어 성경 통합

**헬라어 예시 (신약):**
```
ἀγαπάω (agapaō) - G25
뜻: 사랑하다, 애정을 갖다
형태: 동사, 과거시제, 능동태, 직설법, 3인칭, 단수
```

**히브리어 예시 (구약):**
```
אָהַב (ahav) - H157
뜻: 사랑하다
어근: אהב
```

### 5. 교차 참조 자동 발견

관련 구절을 자동으로 연결:

**예언-성취 연결:**
- 예언: 이사야 7:14 "처녀가 아이를 낳을 것이다"
- 성취: 마태복음 1:22-23 "이는 예언을 이루려 하심이라"

**평행 본문:**
- 오병이어 기적:
  - 마태복음 14:13-21
  - 마가복음 6:30-44
  - 누가복음 9:10-17
  - 요한복음 6:1-15

### 6. 한국어 특화 기능

#### 한자(漢字) 표시
신학 용어의 한자를 보여줌으로써 더 깊은 이해 가능:

```
속죄 (贖罪) - 贖 = 속량하다, 罪 = 죄
구원 (救援) - 救 = 구하다, 援 = 돕다
은혜 (恩惠) - 恩 = 은혜, 惠 = 혜택
믿음 (信)   - 信 = 믿을 신
소망 (所望) - 所 = 바, 望 = 바라다
```

#### 로마자 표기
한국어 발음을 로마자로 표시:

```
원문: 하나님이 세상을 이처럼 사랑하사
로마자: Hananim-i sesang-eul icheoreom saranghasa
```

#### 한국어 최적화 타이포그래피
- 줄 간격: 1.8-2.0 (영어보다 넓음)
- 폰트: 나눔고딕, Noto Sans KR
- 최소 글자 크기: 16px
- 모바일 최적화


#### ❌ 피해야 할 검색


### 언어 혼용 검색

한국어와 영어를 섞어서 검색 가능:


## 한국어 특화 기능

### 1. 한자(漢字) 표시 기능

#### 왜 필요한가?

많은 한국어 신학 용어는 한자어입니다. 한자를 보면 단어의 깊은 의미를 이해할 수 있습니다.

#### 사용 방법

1. **설정**에서 "한자 표시" 켜기
2. 신학 용어에 한자 자동 표시

#### 예시

```
일반 표시: 속죄
한자 표시: 속죄 (贖罪)
          ↓
          贖 = 속량하다 (돈 주고 사다)
          罪 = 죄

일반 표시: 구원
한자 표시: 구원 (救援)
          ↓
          救 = 구하다
          援 = 돕다, 지원하다
```

#### 주요 신학 용어 한자

| 한글 | 한자 | 영어 | 원어 |
|------|------|------|------|
| 속죄 | 贖罪 | Atonement | כָּפַר (H3722) |
| 구원 | 救援 | Salvation | σωτηρία (G4991) |
| 은혜 | 恩惠 | Grace | χάρις (G5485) |
| 믿음 | 信 | Faith | πίστις (G4102) |
| 의 | 義 | Righteousness | δικαιοσύνη (G1343) |
| 성령 | 聖靈 | Holy Spirit | πνεῦμα ἅγιον |
| 회개 | 悔改 | Repentance | μετάνοια (G3341) |
| 구속 | 救贖 | Redemption | ἀπολύτρωσις (G629) |

### 2. 로마자 표기 기능

#### 누구를 위한 기능인가?

- 한국어를 배우는 외국인
- 한글을 읽을 수 없는 분들
- 발음 연습을 위한 학습자

#### 사용 방법

1. **설정**에서 "로마자 표기" 켜기
2. 한글 아래 로마자 자동 표시

#### 예시

```
원문: 하나님이 세상을 이처럼 사랑하사
로마자: Hananim-i sesang-eul icheoreom saranghasa

원문: 독생자를 주셨으니
로마자: dokssaengja-reul jusyeoss-euni

원문: 그를 믿는 자마다
로마자: geu-reul midneun jamada
```

### 3. 한국어 타이포그래피 최적화

#### 왜 중요한가?

한글은 영어보다 시각적으로 복잡한 글자입니다. 읽기 편하게 하려면 특별한 최적화가 필요합니다.

#### 최적화 내용

**줄 간격 (Line Height):**
- 영어: 1.5
- 한글: 1.8-2.0 ← **더 넓음**

**폰트:**
- 나눔고딕: 한글 전용 최적화 폰트
- Noto Sans KR: Google의 한글 폰트

**글자 크기:**
- 영어 최소: 14px
- 한글 최소: 16px ← **더 큼**

#### 비교

```
영어 (최적화 전):
For God so loved the world
(줄간격 1.5, 14px)

한글 (최적화 전):
하나님이세상을이처럼사랑하사
(줄간격 1.5, 14px) ← 읽기 어려움!

한글 (최적화 후):
하나님이 세상을 이처럼 사랑하사

(줄간격 1.8, 16px, 나눔고딕) ← 읽기 편함!
```

### 4. 존댓말 처리

한국어는 높임말이 중요합니다. 시스템은 하나님과 성경 인물에 대해 적절한 존댓말을 사용합니다.

**예시:**
```
✅ 하나님께서 (주격 높임)
✅ 말씀하시다 (동사 높임)
✅ 드리다 (겸양 동사)

❌ 하나님이
❌ 말하다
❌ 주다
```

## 자주 묻는 질문

### Q1: 검색 결과가 정확한가요?

**A:** 네, 매우 정확합니다. 우리는 `multilingual-e5-large` 모델을 사용하는데, 이는 한국어 의미 이해에 최적화되어 있습니다. 테스트 결과 95% 이상의 정확도를 보입니다.

### Q2: 어떤 한국어 성경 번역본을 지원하나요?

**A:** 현재 지원:
- 개역개정 (1998년) - 가장 많이 사용
- 개역한글 (1961년) - 전통적
- 새번역 (2004년) - 현대적
- 공동번역 (1977년) - 가톨릭/개신교 공동

추가 번역본도 계속 추가 예정입니다.

### Q3: 검색어를 정확히 입력해야 하나요?

**A:** 아니요! 그게 바로 의미 검색의 장점입니다.

```
예시:
검색: "예수님이 용서에 대해 하신 말씀"
→ "용서"라는 단어가 없어도 관련 구절을 찾습니다

검색: "힘들 때 위로"
→ "위로"라는 정확한 단어가 없어도
  격려하고 힘을 주는 구절들을 찾습니다
```

### Q4: 한자 표시가 왜 필요한가요?

**A:** 한국어 신학 용어의 70% 이상이 한자어입니다. 한자를 보면:
1. 단어의 정확한 의미 이해
2. 관련 개념들의 연결 파악
3. 신학적 깊이 있는 공부 가능

예: "속죄(贖罪)" - 贖(속량) + 罪(죄) = 죄값을 치르고 사다

### Q5: 무료인가요?

**A:** 네, 완전 무료입니다!
- 사용 무료
- 오픈소스 (MIT 라이선스)
- 광고 없음
- 계정 필요 없음

### Q6: 모바일에서도 사용할 수 있나요?

**A:** 네, 모바일 최적화되어 있습니다:
- 반응형 디자인
- 터치 친화적 인터페이스
- 한국어 타이포그래피 최적화
- 빠른 로딩 속도

### Q7: 오프라인에서도 사용 가능한가요?

**A:** 현재는 인터넷 연결이 필요합니다 (AI API 사용). 향후 오프라인 버전 제공 예정입니다.

### Q8: 원어 성경도 검색할 수 있나요?

**A:** 네, 헬라어(신약)와 히브리어(구약) 원문도 포함되어 있습니다. 하지만 원어로 직접 검색하기보다는, 한국어나 영어로 검색한 후 원어를 확인하는 것을 추천합니다.

### Q9: 교회에서 사용해도 되나요?

**A:** 물론입니다! 다음 용도로 자유롭게 사용하세요:
- 주일학교 준비
- 설교 준비
- 성경 공부 모임
- 개인 큐티

## 문제 해결

### 검색이 안 되요

#### 증상
검색 버튼을 눌러도 아무 결과가 안 나옴

#### 해결 방법

1. **인터넷 연결 확인**
   ```
   브라우저에서 다른 사이트 접속 확인
   ```

2. **검색어 확인**
   - 최소 2글자 이상 입력
   - 특수문자만 입력하지 않았는지 확인

3. **번역본 선택 확인**
   - 최소 1개 이상의 번역본 선택되어 있는지 확인

4. **브라우저 새로고침**
   ```
   Ctrl + F5 (Windows)
   Cmd + Shift + R (Mac)
   ```

### 한글이 깨져서 보여요

#### 증상
한글이 □□□ 또는 ??? 로 표시됨

#### 해결 방법

1. **브라우저 인코딩 확인**
   - Chrome: 설정 → 고급 → 언어 → UTF-8 확인

2. **브라우저 캐시 삭제**
   ```
   Chrome: Ctrl + Shift + Delete
   → "캐시된 이미지 및 파일" 삭제
   ```

3. **폰트 문제**
   - 나눔고딕 또는 Noto Sans KR 설치 확인
   - 브라우저가 최신 버전인지 확인

### 한자가 표시 안 돼요

#### 증상
한자 표시를 켰는데도 안 보임

#### 해결 방법

1. **설정 확인**
   - 상단 메뉴 → 설정 → "한자 표시" 체크

2. **브라우저 폰트 확인**
   - 한자 폰트가 설치되어 있는지 확인
   - 브라우저 최소 확대/축소 100% 확인

3. **페이지 새로고침**

### 검색이 느려요

#### 증상
검색 결과가 나오는데 시간이 오래 걸림

#### 해결 방법

1. **정상 속도 확인**
   - 첫 검색: 1-2초 정상
   - 캐시된 검색: 0.5초 미만

2. **인터넷 속도 확인**
   ```
   speedtest.net에서 속도 테스트
   최소 권장: 5Mbps
   ```

3. **서버 상태 확인**
   - 상태 페이지: [링크]
   - 일시적 서버 과부하일 수 있음

### 원어가 이상하게 보여요

#### 증상
헬라어나 히브리어가 깨져서 보임

#### 해결 방법

1. **유니코드 지원 확인**
   - 브라우저가 UTF-8 지원하는지 확인
   - 최신 버전 브라우저 사용 권장

2. **폰트 설치**
   - 헬라어: Gentium Plus, SBL Greek
   - 히브리어: Ezra SIL, SBL Hebrew

3. **브라우저 설정**
   - Chrome: 설정 → 모양 → 글꼴 맞춤설정
   - 세리프: SBL Greek/Hebrew

### 모바일에서 버튼이 안 눌려요

#### 증상
터치해도 반응이 없음

#### 해결 방법

1. **터치 영역 확인**
   - 버튼을 정확히 눌렀는지 확인
   - 확대하여 더 크게 보기

2. **브라우저 호환성**
   - Safari (iOS): 최신 버전 사용
   - Chrome (Android): 최신 버전 사용

3. **화면 회전**
   - 가로/세로 모드 전환 시도
